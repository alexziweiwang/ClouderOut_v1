each "sequence"(small level) can contain 1 or more "pieces/screens"; game-round(?) [sequence is with LinkedList pointers for flexible orders]
each "chapter"(large level) is linked by linear order

editing option of "piece/screen" can be among the following game types: card, board, fort defense, fort offense, text adventure
